# Desktop PC Remote Power Control

Use an ESP32 to remotely power on and off your computer.  
Inspired by https://noisycarlos.com/project/how-to-turn-your-computer-on-and-off-remotely/  

## Usage:

### Parts:

- [ESP32-C3 SuperMini](https://www.aliexpress.com/item/1005007446928015.html)
- [Optocouplers](https://www.aliexpress.com/item/1005006281381268.html)
- 220Ω-600Ω resistors
- [F_PANEL Breakout Cable](https://www.aliexpress.com/item/1005010545909886.html)
- [Prototype Board (6x4cm)](https://www.aliexpress.com/item/1005010400535614.html)
- [Dupont Jumper Wires (M-F)](https://www.aliexpress.com/item/4000203371860.html)
